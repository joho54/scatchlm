Anatomy: 3-Layer Glass Construction

Layer 1: Glass Body

Frosted base with high blur and saturation boost. Saturation >100% is key — Apple uses it to make background colors bleed through vividly.

background: rgba(255,255,255,0.08); backdrop-filter: blur(40px) saturate(180%);

Layer 2: Refraction Border (::before)

Gradient border using mask-composite trick. Warm tones (red/orange) at light-entry, cool tones (blue) at shadow-side. This is the "glass edge catching light" effect.

background: linear-gradient( 135deg, rgba(255,255,255,0.6) 0%, rgba(255,200,150,0.3) 15%, rgba(255,255,255,0.15) 40%, rgba(150,200,255,0.2) 70%, rgba(255,255,255,0.05) 100% ); /* mask-composite: exclude */

Layer 3: Inner Glow (::after)

Directional highlight from light source + inset box-shadow for glass thickness. Top-left bright, bottom-right dim.

background: linear-gradient( 135deg, rgba(255,255,255,0.12) 0%, transparent 40% ); box-shadow: inset 0 1px 1px rgba(255,255,255,0.15), inset 0 -1px 1px rgba(0,0,0,0.05);
