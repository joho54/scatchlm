## 1. 에브리타임 (정보글 톤, 학생 타겟)

**제목: 손으로 쓰면서 공부하는데 AI가 바로 피드백 주는 아이패드 앱 만들었어요 (ScatchLM)**

```
필기하면서 공부하는 사람들 보세요.

아이패드 + 애플펜슬로 손글씨 쓰면, AI가 그걸 읽고
바로 피드백을 주는 학습 앱 'ScatchLM'을 만들었습니다.

- 외국어(고전어 포함), 전공 공부, 기술 심화 등 범용으로 씀
- 손으로 정리 → 즉시 AI 피드백 받는 학습 루프가 핵심
- 교재 PDF를 연결하면 그 페이지/챕터 내용 기준으로
  피드백이랑 질문 답변을 해줌 (그냥 일반 챗봇이랑 다른 점)

타이핑 말고 '손으로 쓰면서' 머리에 남기는 공부 좋아하면
잘 맞을 거예요.

현재 무료로 받아볼 수 있고, 버전 1.0.1입니다.
앱스토어에 'ScatchLM' 검색!

써보고 피드백 주시면 반영할게요 🙏
```
